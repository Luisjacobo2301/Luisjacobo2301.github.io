Fuselage FEA Analysis

February 2024 - April 2024 Advanced Simulation Project

Project Overview

Created a detailed Finite Element Analysis of a commercial aircraft fuselage using ABAQUS. This project involved simulating various loading conditions to analyze structural integrity and developing optimization strategies for improved performance and weight reduction.

Key Responsibilities:

  • Developed complex FEA models using ABAQUS
  • Simulated multiple loading scenarios for structural analysis
  • Created comprehensive technical documentation
  • Proposed structural optimization solutions
