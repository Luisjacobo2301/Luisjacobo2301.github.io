FDM Structural Analysis

January 2024 - Present Research Project

Project Overview

This research project focuses on developing a novel method to predict delamination in FDM 3D printed models. By combining Finite Element Modeling, NASMAT, Python, and the generalized Method of cells, we aim to enhance the understanding and prediction of structural behavior in 3D printed components.

Key Responsibilities:

  • Developed computational models using Python and NASMAT
  • Implemented Finite Element Analysis for structural predictions
  • Created algorithms for delamination prediction
  • Validated models through experimental testing
